GetWorldInfo App Urls

Home

http://getworldinfo.webdeveloperjourney.xyz/home

This url gives list of all continents like asia,africa,america,europe and oceania.
All Countries

http://webdeveloperjourney.xyz/allCountry/region/:continentName

This url gives all countries of continent.The Continent which is get selected is in route parameter 'continentName' that user will select in home page.According to selection of continent, all countries will get displayed with filters in this page.
Single Country Info

http://webdeveloperjourney.xyz/Country/:countryName

This url gives details of a country.The country which is get selected is in route parameter 'countryName' that user will select in all country page.
